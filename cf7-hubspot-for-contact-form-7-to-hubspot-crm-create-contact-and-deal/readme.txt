This plugin integrates HubSpot forms with Contact Form 7 forms.

== Description ==

This is a FREE plugin which seamlessly integrates HubSpot contacts and deals with Contact Form 7 forms.

Once the plugin is activated you will have the option to enable/disable the form integration. In enabled mode, the status code and response from HubSpot will be stored for the last form submission so you can debug if there are any issues.

Note: This plugin uses the PHP library cURL. Please make sure that the cURL library is enabled on your server.

== Installation ==

1. Sign in to your WordPress site as an administrator.
2. In the main menu go to Plugins > Add New.
3. Search for "CF7 HubSpot For Contact Form 7 to HubSpot CRM create Contact and Deal" (without the quotes of course) and click install.
4. After the plugin is installed activate the plugin.
5. That's it. Go ahead and start creating forms using Contact Form 7 and you will see a tab called "HubSpot Contact/Deal Integration".
